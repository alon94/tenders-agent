# Handoff — גרסת מובייל: לוח מכרזים (Tenders Agent)

## Overview
מפרט מימוש **מובייל** של אפליקציית "שווה מכרזים / מועדון עסקים 360" (מבוסס `tenders-agent`), בכיוון עסקי־מאופק ובפלטת **תכלת/כחול**. RTL מלא. המסמך מכסה את **כל הדפים הפנימיים** בזרימת המובייל: רשימת מכרזים, מכרז בודד, מכרזים מסומנים, סוכן חכם, וערבויות וליווי.

## About the Design Files
הקבצים הם **ייחוס עיצובי ב-HTML** (פרוטוטיפ) — לא קוד production. יש **לשחזר את העיצוב בקודבייס הקיים** (React ככל הנראה) לפי מערכת הרכיבים והדפוסים שם. `tenders-mobile-standalone.html` הוא בנדל עצמאי לתצוגה/iframe; `Tenders Dashboard.dc.html` הוא קוד המקור עם הערכים המדויקים והנתונים.

התיקייה `screens/` מכילה צילומי מסך של המסכים.

## Fidelity
**High-fidelity.** לשחזר פיקסל-קרוב. Mobile-first, RTL.

## Mobile Frame & Layout Rules
- **Viewport מטרה:** 390×844 (iPhone). הפריסה נוזלית — להשתמש ב-`100%`/`flex`/`min-width:0`, **לא** ברוחבים קבועים. (הבאג המקורי נבע מגריד דסקטופ ברוחב קבוע שנדחס במובייל.)
- **מבנה מסך:** `flex-direction: column` בגובה מלא — Header קבוע למעלה, אזור תוכן `flex:1; overflow-y:auto`, וסרגל תחתון קבוע (ניווט או פעולות).
- **Safe area:** padding תחתון ~22px בסרגלים התחתונים (מחליף `env(safe-area-inset-bottom)`).
- כותרות מכרז חייבות לזרום (`line-height:1.45`, `text-align:right`) — לעולם לא רוחב קבוע שגורם לשבירת אות-אות.

## Screens (כל הדפים הפנימיים)

### 1. רשימת מכרזים (גילוי מכרזים) — מסך בית
- **Header לבן:** שורת סטטוס (שעה + אינדיקטורים), שורת מותג (אווטאר לוגו 34 כחול, כותרת "גילוי מכרזים" + תת "2,988 פעילים · עודכן HH:MM", אייקון פעמון עגול), שדה חיפוש (`#f4f6f8`, רדיוס 12, אייקון ⌕ + מבטא ✦ כחול).
- **KPI strip:** גלילה אופקית (`overflow-x:auto`), כרטיסי min-width 96 (`#f6f8fa`, גבול, רדיוס 12): נקודת צבע + מספר 22px + תווית 10.5px. ערכים: 2,988 פעילים · 16 נסגרים בשבוע · 0 חדשים · 2,980 מוצגים.
- **Filter chips:** גלילה אופקית, pills: "הכל · 2,980" (פעיל תכלת), "נסגרים · 16", "חדשים · 0".
- **רשימת כרטיסים** (גלילה אנכית): כל כרטיס — שורת תגיות סטטוס + ציון (מספר 22px + פס band) למעלה, כותרת זורמת, מטא "‹מפרסם› · פורסם ‹תאריך›", footer (קו עליון) עם "הגשה עד ‹תאריך› · ‹ימים›" + כפתור "לפרטים" כהה.
- **Bottom nav** (5 טאבים): גילוי / מסומנים / סוכן / ערבויות / עוד. הפעיל בכחול `#2b6fc4` משקל 700, השאר `#8a97a3`.

### 2. מכרז בודד (פרטי מכרז)
- **Header:** חץ חזרה (→ ב-RTL), "פרטי מכרז", כוכב שמירה כחול.
- **כרטיס ציון hero:** ריבוע 64 תכלת עם "75" + "ציון התאמה", לצידו תגיות סטטוס + "‹מפרסם› · מכרז ‹מס'›".
- **כרטיס כותרת:** כותרת מלאה + שורת 3 עמודות (פורסם / הגשה עד / נותרו).
- **כרטיס תובנת סוכן:** רקע `#f0f6fd` גבול `#cfe0f4`, כותרת "✦ תובנת הסוכן החכם" כחולה + פסקה.
- **Sticky action bar:** "☆ שמירה" (משני) + "למכרז המלא ↗" (כחול, flex:1).

### 3. מכרזים מסומנים
- **Header:** "מכרזים מסומנים" + "N מכרזים שמורים" + כוכב תכלת; שני filter pills (הכל / נסגרים בקרוב).
- **כרטיסים:** כותרת + ציון בצד, מטא "‹מפרסם› · הגשה עד ‹תאריך› · ‹ימים›", שורת פעולות: "✦ החלט עבורי" (משני) · "לפרטים" (תכלת) · "✕" (הסרה, אדום).
- Bottom nav — טאב "מסומנים" פעיל.

### 4. סוכן חכם
- **Header:** אווטאר ✦ כחול + "הסוכן החכם" + "● פעיל · סורק N מכרזים".
- **תוכן (סגנון צ'אט):** בועת סוכן (רדיוס `16px 16px 16px 4px`, יישור התחלה); כרטיס **תהליך עבודה** — צעדים ממוספרים עם עיגול סטטוס (הושלם = תכלת ✓ `#e8f1fb`/`#1e5aa8`, פעיל/ממתין = צהוב `#f5efdc`/`#8a6d1f`): איתור מכרזים → סינון לפי פרופיל → דירוג והתאמה → הכנת סיכום החלטה; בועת תוצאה תכלת עם שאלת המשך.
- **Input bar:** שדה pill "שאל את הסוכן…" + כפתור שליחה עגול כחול (↑).

### 5. ערבויות וליווי
- **Header:** "ערבויות וליווי" + תת-כותרת.
- **Summary band:** שני כרטיסים — "₪420K סה"כ ערבויות פעילות" · "2 פגות תוקף החודש" (מספר אדום).
- **רשימת ערבויות:** כרטיס — כותרת מכרז + סכום (משמאל), מטא "‹סוג ערבות› · תוקף עד ‹תאריך›", footer: תגית סטטוס (פעילה=תכלת / פגה בקרוב=אדום) + "ניהול ›" כחול.
- CTA כהה "+ בקשת ערבות חדשה".
- Bottom nav — טאב "ערבויות" פעיל.

## Interactions & Behavior
- **Bottom nav:** מעבר בין 5 המסכים; טאב פעיל מודגש בכחול.
- **חיפוש:** שדה חופשי → סינון הרשימה. מבטא ✦ פותח חיפוש AI.
- **צ'יפים:** מחליפים קבוצת תוצאות (הכל / נסגרים / חדשים).
- **כרטיס מכרז:** קליק → מסך מכרז בודד. "☆" toggle שמירה (מזין את "מסומנים").
- **מכרזים מסומנים:** "✕" מסיר מהשמורים; "החלט עבורי" מפעיל את הסוכן על אותו מכרז.
- **סוכן חכם:** צעדי תהליך מתקדמים (הושלם→פעיל→ממתין); שליחת הודעה מוסיפה בועה ומריצה שלב.
- **ערבויות:** "ניהול ›" פותח פרטי ערבות; "בקשת ערבות חדשה" פותח טופס.
- **Empty states:** "0 חדשים" / אין מסומנים / אין ערבויות — להציג מצב ריק מנוסח.

## State Management
- `activeScreen`: discover | saved | agent | guarantees | more
- `filters`: `{ domain, publisher, tab: all|closingThisWeek|newThisWeek }`
- `search`: מחרוזת · `savedTenderIds`: מסומנים (toggle) · `selectedTenderId`: למסך הבודד
- `agent`: `{ status, steps[], messages[] }` · `guarantees[]`: `{ title, type, expiry, amount, status }`
- `tenders[]`: `{ title, publisher, published, due, daysLeft, score(0–100), tags[], status }` — נשלף מ-data.gov.il/API; KPI נגזרים מהרשימה.

## Assets
- פונט **Assistant** (Google Fonts) — מוטמע בבנדל.
- אין תמונות. אייקונים = תווי יוניקוד (⌕ ☆ ◈ ◵ ❏ ✦ ↑ ✓ ✕ 🔔 →) — להחליף באייקוני מערכת הרכיבים (lucide/heroicons).
- לוגו — אות "ש" על ריבוע כחול; להחליף בלוגו המותג האמיתי.
